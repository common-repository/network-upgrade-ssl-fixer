With Network Upgrade SSL fixer Say good bye to encryption errors!
== Description ==
if you are experiencing errors like  
* stream_socket_client(): SSL operation failed with code 1. OpenSSL Error messages: error:14077410:SSL routines:SSL23_GET_SERVER_HELLO:sslv3 alert handshake failure 
* stream_socket_client(): Failed to enable crypto stream_socket_client(): unable to connect to ssl://anything.net:443 (Unknown error)" Error when your trying to upgrade your network .
this plugin will help you out to solve the problem.
== Installation ==
1. Upload `Network-Upgrade-Fixer.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
